# TASTY TALES API

## Overview

A simple CRUD API for managing recipes using Node.js, Express, and MongoDB. Implemented this API with MVC architecture.

## Tech Stack

- Node.js
- Express.js
- MongoDB with Mongoose
- Postman for API documentation

## API Endpoints

- `POST /api/v1/recipes` - Create a recipe
- `GET /api/v1/recipes` - Get all recipes
- `GET /api/v1/recipes/:id` - Get a recipe by ID
- `PUT /api/v1/recipes/:id` - Update a recipe by ID
- `DELETE /api/v1/recipes/:id` - Delete a recipe by ID

For more information refer Postman documentation (inside data folder)
